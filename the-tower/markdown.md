# 서버 명령어 모음

/home\
설정된 귀환지로 이동합니다. 저장된 귀환지가 1개인 경우, 해당 명령어만 입력해도 저장된 귀환지로 이동합니다.\
2개 이상이 등록된 경우, /home 저장한이름을 입력하여 해당 위치로 이동하실 수 있습니다.\
2개 이상이 등록된 경우, /home만 입력하면 관련 설정이 가능합니다.\
최대 4개까지 등록 가능합니다.\
3번째, 4번째 귀환지 등록시 100만원이 필요합니다.

&#x20;/party\
경험치 공유용 파티입니다. (4인제한)\
경험치획득량 : 경험치X1.25/인원수

&#x20;/파티 만들기\
보스레이드용 파티를 생성합니다.\
&#x20;/파티 정보\
파티에 포함 된 인원의 정보를 확인하실 수 있습니다.\
&#x20;/파티 초대 닉네임\
파티에 새로운 플레이어를 초대하실 수 있습니다.\
&#x20;/파티 탈퇴\
파티에서 탈퇴합니다.

\
&#x20;/qs create 가격\
상자를 바라보고 판매를 하고 싶은 아이템을 들고 입력하시면 판매할 가격으로 유저 거래용 상점이 생성됩니다.\
(팔 아이템을 들고 상자를 클릭해서도 생성하실 수 있으나, 일부 아이템은 상자를 좌클릭할 수 없는 아이템이 있습니다. 그런 경우에 사용하시면 됩니다.

/qs finditem TO\_BUY 찾는아이템\
/qs finditem TO\_SELL 찾는 아이템\
유저 거래용 상점을 검색할수 있는 명령어입니다.\
검색할때는 찾는아이템의 한글이름 또는 아이템의 영문명으로 찾을수 있습니다
