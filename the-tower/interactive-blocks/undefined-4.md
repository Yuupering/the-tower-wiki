# 팔찌

조합법에는 유물 조각 1개, 고대 유물의 파편 10개, 제련된 오리하르콘 총 6개, 강력한 야생의 기운 총8개, 10만코인 10개가 필요합니다.

1. 감각적인 모험가의 팔찌 (+n강) \
   스킬 치명타 추가 피해: +10% (+5n%) \
   아이템 레벨: +30 (+50n)\
   ![](<../../.gitbook/assets/image (23).png>)<br>
2. 강인한 모험가의 팔찌 (+n강) \
   최대 체력: +150 (+75n) \
   아이템 레벨: +30 (+50n)\
   ![](<../../.gitbook/assets/image (24).png>)<br>
3. 난폭한 모험가의 팔찌 (+n강) \
   공격 속도: +0.4 (+0.4n) \
   아이템 레벨: +30 (+50n)\
   ![](<../../.gitbook/assets/image (25).png>)<br>
4. 날카로운 모험가의 팔찌 (+n강) \
   무기 피해량: +3% (+2.5n) \
   아이템 레벨: +30 (+50n)\
   ![](<../../.gitbook/assets/image (26).png>)<br>
5. 도전적인 모험가의 팔찌 (+n강) \
   스킬 쿨타임 감소: +3% (+2n) \
   아이템 레벨: +30 (+50n)\
   ![](<../../.gitbook/assets/image (27).png>)<br>
6. 지혜로운 모험가의 팔찌 (+n강) \
   마나 회복량: +0.5 (+1n) \
   아이템 레벨: +30 (+50n)\
   ![](<../../.gitbook/assets/image (28).png>)\
   <br>
7. 통달한 모험가의 팔찌 (+n강) \
   스킬 피해량: +4% (+4n%) \
   아이템 레벨: +30 (+50n)\
   ![](<../../.gitbook/assets/image (29).png>)<br>
8. 포악한 모험가의 팔찌 (+n강) \
   무기 치명타 추가 피해: +5% (+5n%)\
   아이템 레벨: +30 (+50n)\
   ![](<../../.gitbook/assets/image (30).png>)<br>
9. 현명한 모험가의 팔찌 (+n강) \
   최대 마나: 150 (+100n) \
   아이템 레벨: +30 (+50n)\
   ![](<../../.gitbook/assets/image (31).png>)<br>
10. 활동적인 모험가의 팔찌 (+n강) \
    체력 회복량: +1 (+1.5n) \
    아이템 레벨: +30 (+50n)\
    ![](<../../.gitbook/assets/image (32).png>)
