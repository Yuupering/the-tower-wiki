# 도감



필요한 아이템을 소지 한 뒤, 도감내의 아이템을 클릭하면 아이템이 소모되며, 증표 강화 재료를 줍니다.\
\
증표 강화 재료는 개인 귀속아이템이며, 공유를 시도하다 적발되면 제제받을 수 있습니다.\
\
도감은 일정 진척도를 달성할때 마다 추가적인 보상을 받을수 있습니다.<br>
