# 어부

첫 특성을 어부로 선택한 경우, 어부 전용 낚싯대가 1회 한정 지급됩니다<br>

1차 :\
특정 등급의 물고기를 낚았을 때 서버 돈이 즉시 지급됩니다.\
브론즈 : 25 / 실버 : 100 / 골드 : 250 / 다이아 : 750 / 플레티넘 : 7500\
전설이상의 물고기 획득 시, 신비한 기운의 물고기를 즉시 8개 획득합니다.

어부 전용으로, 신비한 기운의 물고기의 획득 확률은 어부 행운아 관련 업적으로 강화할 수 있습니다.\
어부 일때 기본 8%로적용(타 특성인 경우 4%)\
행운아 - 4%p 상승, 전설적인 행운아 - 4%p 상승, \
신화적인 행운아 - 실버등급 물고기에도 확률 적용

2차 : \
특정 등급 물고기를 낚을 때 확률로 네더의 별이 추가 드롭됩니다. - 소비값 없음(패시브)\
해당 패시브는 직업 차수가 증가할수록 강화됩니다. (각 2차 / 3차 / 4차 / 에메랄드 순 )\
골드 물고기 : 0.04% / 0.1% / 0.5% / 1%\
다이아몬드 물고기 : 0.5% / 1% / 2% / 3%\
플레티넘 물고기 : 2% / 3% / 5% / 15%

3차 : 피버 타임! (20분 지속, 스태미나 10 소모)\
&#x20;      1차의 돈지급의 효과가 1.5배로 강화됩니다 (에메랄드 등급 달성 시 1.5배 ⇒ 2배로  강화됩니다.)

4차 : \
20분동안 신화등급 물고기를 낚을 확률이 생성됩니다(0.05%), 스킬을 사용한 이후 낚시를 하다 0.025%/0.075%(에메랄드) 확률로 랜덤한 확률의 전설 \~ 신화 물고기 랜덤 박스 토큰을 흭득합니다, (랜덤 박스 구성품 낚시꾼의 혼 41.95%, 전설 57%, 하늘의 성소 신화 0.7%, 지옥의 성소 신화 0.35%)\
(스태미나 10 소모, 지속시간 20분)
