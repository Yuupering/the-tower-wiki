# 농부

첫 특성을 농부로 선택한 경우, 농부 뉴비 전용 괭이, 7x7 1개분 땅, 허수아비, 스프링쿨러 2개, 10만원이 1회 한정 지급됩니다<br>

1차 : 인벤토리에 작물 아이템이 있을때 작물 채집시 자동 심어지는 기능 해금\
농부 특성을 보유한 경우, 대지의 기운이 담긴 열매 드롭율이 상승합니다(2%=>10%)\
모든 차수 커스텀 농사를 기본 7x7크기 2구역을 사용 가능합니다.\
(해당 땅 제한은 농부 업적을 클리어해서 얻는 징표로 3구역까지 확장 가능하며, 이는 농부 특성만 도전과제 도전이 가능합니다.)\
특수 작물을 수확 할 때 0.7% 확률로 금등급 작물을 수확 합니다.\
특수 작물을 수확 할 때 25%의 확률로 한번 더 수확합니다.\
추가 수확 시 0.7%의 확률로 금 작물을 획득합니다.&#x20;

2차 : 기본 작물 수확시 5분동안 10% 확률로 2배 수확 - 스태미나 소비 20\
&#x20;        지속시간 중 대지의 기운이 담긴 열매 드롭률 3% 추가.\
&#x20;        지속시간 중 달빛의 기운이 서린 괭이 효과를 부여.\
&#x20;        (일반 작물 수확 시 추가로 무작위 커스텀 작물 1회 수확하는 효과 발생)\
\
3차 : \[패시브] [커스텀 크롭 작물](../undefined/undefined-1.md) 수확시 10분동안 10% 확률로 3배 수확\
&#x20; 3차 스킬 시도 성공 시 5%의 확률로 고대 유물의 파편을 습득 가능 합니다.\
\
4차 : 기본 작물 수확시 0.02% 로 농부의 소중한 당근 코어 흭득\
&#x20;-농부의 소중한 당근 코어 아이템은 유물 "농부의 소중한 당근"아이템 제작 및 스테미나 포션, 작물 과 성장 영양제 - 얼티밋등에 활용하실 수 있습니다.\
![](../../../.gitbook/assets/image.png)\
작물 과 성장 영양제 - 얼티밋은 작물이 심어져 있지 않은 땅에 우클릭 하여 사용 가능하며 일정시간동안 과성장 작물로 성장할 확률이 10%로 증가합니다.

에메랄드 등급 달성 시 커스텀 작물 수확 시, 1% 확률로 금등급 작물이 추가 드롭됩니다.
