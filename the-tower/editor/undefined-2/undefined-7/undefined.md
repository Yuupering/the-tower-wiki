# 개화



파괴의 합주곡\
&#x20; -랩소디 스킬 시전시 이동 속도 버프가 삭제됩니다.\
&#x20; -랩소디 스킬 발동시 20칸 이내의 자신 포함 모든 플레이어의 스킬 피해량 스텟이 10% 증가됩니다.\
&#x20; -발동 이후 20초 뒤에 주변 20칸 이내의 모든 플레이어의 마나가 40% 즉시 회복됩니다.\
&#x20; -랩소디 스킬 발동시 지속 시간 동안 시전자 본인의 최대 마나량이 소폭 증가됩니다.\
&#x20; -해당 스킬은 주변 음유시인과 겹칠수 없습니다.



프리즘 콘체르토\
&#x20; -최대 마나 -1250\
&#x20; -착용 시 천사의 축복 기존 효과가 사라지며 쿨타임과 마나 소비가 제거됩니다.\
&#x20; -엑스트라 스킬의 순서가 변경됩니다: 힐) 마나 회복) 피해량 증가) 프리즘 콘체르토\
&#x20; -천사의 축복을 발동했을 때 5초 이내 특정 스킬 키를 입력 횟수에 맞게 입력하면 각 효과가 발동됩니다. 입력 실패 시 체력과 마나가 줄어듭니다.\
\
&#x20; -비바체 (내부 쿨타임 10초): 사운드 웨이브 5번 입력\
&#x20; -주변 아군을 천사의 축복 회복량의 2배만큼 회복시키며 5초 동안 저항 1 버프를 부여합니다.\
&#x20; -발동 시 도발 수치가 누적됩니다.\
\
&#x20; -디스트로이드 레퀴엠 (내부 쿨타임 1초): 사운드 웨이브 2, 바이브레이션 1, 파괴의 교향곡 2 입력 전방으로 음표 투사체를 발사합니다.\
&#x20; -멜로디 오브 라이트 (내부 쿨타임 10초): 랩소디 2. 천사의 축복 3 입력\
&#x20; -10초 동안 자신 제외 주변 아군에게 물리 피해량 +10%, 스킬 피해량 +10% 버프를 부여합니다. 발동 시 도발 수치가 누적됩니다.\
&#x20; -디스트로이드 레퀴엠 발동 성공시 10초간 유지 되는 강화 사운드 웨이브의 피해량 +10 증가 버프 부여합니다.\
&#x20;  (10초동안 갱신이 없으면 즉시 스택이 0으로 조정됩니다.)

\
\
&#x20; -마나 소나타 (내부 쿨타임 5초): 천사의 축복 4, 랩소디 1 입력\
&#x20; -천사의 축복 회복량에 비례해 자신 제외 주변 아군의 마나를 회복시킵니다.\
&#x20; -발동 시 도발 수치가 누적됩니다.
