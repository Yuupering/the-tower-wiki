# 전투 직업

1. 레벨링 시스템을 채용하고 있습니다.  \
   \- 몬스터를 처치 하시거나, 생활 직군에 관련한 행동을 하실시 RPG 레벨업이 가능합니다.\
   레벨 30에 아이언 등급, 레벨 60에 골드 등급 달성으로 상승합니다.<br>
2. 맥시멈 레벨은 120레벨이며 100레벨부터는 익스피리언스 카탈리스트 아이템을 사용하여 레벨링이 가능합니다. 익스피리언스 카탈리스트 아이템이 없다면 극한의 노가다를 요구합니다.<br>
3. 100레벨까지는 레벨업당 <mark style="color:yellow;">**4 스텟 포인트, 1 스킬 레벨업 포인트**</mark>가 지급됩니다.<br>
4. 스텟 포인트로는 다음과 같은 스텟을 올릴수 있습니다.

<figure><img src="../../../.gitbook/assets/캡처 (27).PNG" alt=""><figcaption></figcaption></figure>

왼쪽부터 물리 피해량 증가, 스킬 피해량 증가, 최대 체력/체력 리젠량 증가, 최대 마나/마나 리젠량 증가, 물리크리티컬 피해량 증가,  물리 크리티컬 확률 증가, 스킬 크리티컬 확률 증가, 스킬 크리티컬 피해량 증가입니다.

5. 스킬  레벨업 포인트를 투자하여 스킬의 레벨을 증가 시킬수 있습니다.

<figure><img src="../../../.gitbook/assets/캡처 (29).PNG" alt=""><figcaption></figcaption></figure>

초록색 유리판에 마우스를 올려 현재 잔여 스킬 포인트를 확인 할 수 있고 올리고 싶은 스킬 아이콘을 클릭하신 이후 초록색 유리판을 클릭하시면 스킬이 강화됩니다. 초록색 염료에 마우스를 올릴 경우 다음 레벨을 확인 할 수 있습니다.

5. 만렙을 찍는다 해도 모든 스킬과 모든 스텟을 찍을수 없습니다! 신중하게 투자해주세요.\
   하지만 스텟 초기화 책과 스킬 초기화 책으로 초기화 하여 다시 찍을 수 있습니다.  <br>
6. 만렙 기준 <mark style="color:yellow;">**396 스텟 포인트, 99 스킬 레벨업 포인트**</mark>가 지급됩니다.<br>
7. 스텟 포인트책은 존재하지 않습니다. 그러나스킬 포인트책은 파밍이 가능합니다.\
   \- 많은 노가다가 요구됩니다.<br>
8. 100레벨 이후 초월 레벨로 돌입합니다. 1레벨당 추가 고유 스텟을 흭득합니다.

<figure><img src="../../../.gitbook/assets/캡처 (30).PNG" alt=""><figcaption></figcaption></figure>

아르카나 직업군은 초월자의 힘 비례하여 스킬 공격력 강화 효과를 흭득합니다.\
&#x20; \- 아르카나 직업군은 스킬 쿨 타임 감소 스텟에 영향을 받지 않습니다.
