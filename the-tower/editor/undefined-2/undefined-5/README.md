# 아처

<figure><img src="../../../../.gitbook/assets/캡처 (33).PNG" alt=""><figcaption><p>본 사진의 아이콘과 스킬 설명 순서는 일치하지 않습니다.</p></figcaption></figure>

포지션 : 안정적인 원딜러\
직업 난이도 3/5\
조작 난이도 2/5\
체력 4/5\
딜 2/5

&#x20;

패시브 명사수의 눈빛\
&#x20; \- 해당 스킬은 장착하지 않아도 발동됩니다.\
&#x20; \- 세번째 화살을 적중시켰을 경우 3초동안 저항1 버프를 얻습니다.

&#x20;

1스킬 속사\
&#x20; \- 타겟을 향해 마법 화살을 날립니다.\
&#x20; \- 피해량 계수 : ((피해량)+(명사수의 눈빛 레벨/2))\*스킬 피해량 스텟\
&#x20; \- 레벨당 피해량 : +3.5

&#x20;

2스킬 백스탭\
\- 후방으로 높게 도약합니다\
\- 점프 + 스킬을 사용할경우 더 멀리 뒤로 도약합니다\
\- 바로 속사를 사용 할 경우 (속사x0.4배)+해당 스킬의 피해량만큼 연사가 발동됩니다\
\- 레벨당 피해량: +4

&#x20;

2-1스킬 연사\
&#x20; \- 백스탭 지속시간동안 속사를 사용할경우 연사 콤보가 발동됩니다.\
&#x20; \- 해당 콤보는 속사의 계수를 따라갑니다.\
&#x20; \- 연사 콤보 피해량 계수 : ((피해량)\*1.1+(백스탭 레벨\*4)+(명사수의 눈빛 레벨/2))\*스킬 피해량 스텟\
&#x20; \- 사용 시  자연력 스택(자연의 공명 내용 참조)을 즉시 5 획득합니다.

&#x20;

3스킬 자연의 공명\
&#x20; \- 해당 스킬의 지속시간동안 궁극기 드래곤 피어의 후속타를 사용 못하게 변경됩니다.\
&#x20; \- 자연의 마나와 공명하여 마법의 활을 만들어냅니다\
&#x20; \- 스킬 발동시 콤보 스킬을 사용 할 수 있습니다\
&#x20; \- 자연의 공명 발동시 백스탭 스킬이 1회 충전됩니다.(해당 스킬은 백스탭 스킬과 별개 쿨타임으로 작동되며, 내부 쿨타임은 12초 입니다.)



3-1 스킬 자연의 공명 - 백스탭\
&#x20; \- 자연의 공명 사용 중 백스탭 스킬이 자연의 공명 - 백스탭 스킬로 대체됩니다.\
&#x20; \- 자연력 10을 소모합니다.\
&#x20; \- 자연의 공명 - 백스탭 으로도 연사 콤보를 발동 시킬 수 있지만, 해당 발동으로 자연력이 회복되지는 않습니다.\
&#x20; \- 내부 쿨타임은 12초입니다.

&#x20;

3-2 스킬 자연의 공명 - 속사\
&#x20; \- 자연의 공명 발동 이후 속사를 사용해야 발동 할 수 있습니다.\
&#x20; \- 효과 지속시간동안 속사 스킬을 1초에 한번 사용 할 수 있습니다.\
&#x20; \- 해당 스킬은 속사 스킬의 계수를 따라갑니다\
&#x20; \- 자연의 공명 + 속사 피해량 계수 : ((속사피해량\*0.9)+(자연의 공명 레벨\*4)+(명사수의 눈빛 레벨/2))\*스킬 피해량 스텟\
&#x20; \- 자연의 공명 - 속사 사용 시마다 자연력 스택을 2 획득합니다.

&#x20;

3-3 스킬 자연의 공명 - 데들리 자밸린\
&#x20; \- 자연의 공명 발동 이후 데들리 자밸린을 사용해야 발동 할 수 있습니다.\
&#x20; \- 자연력 10스택을 소모합니다.\
&#x20; \- 3초에 한번 자연의 공명 - 데들리 자밸린 스킬을 사용 할 수 있습니다.\
&#x20; \- 해당 스킬은 데들리 자밸린 계수를 따라갑니다\
&#x20; \- 자연의 공명 - 자밸린 피해량 계수 : ((데들리 자벨린 피해량\*0.45)+(자연의 공명 레벨\*4))\*스킬 피해량 스텟



3-4 스킬 자연의 공명 - 드래곤 피어\
&#x20; \- 자연의 공명 발동 이후 드래곤 피어를 사용해야 발동 할 수 있습니다.\
&#x20; \- 자연력 10스택을 소모합니다.\
&#x20; \- 3초에 한번 자연의 공명 - 드래곤 피어 스킬을 사용 할 수 있습니다.\
&#x20; \- 해당 스킬은 드래곤 피어 계수를 따라갑니다\
&#x20; \- 자연의 공명 - 드래곤 피어 계수 : ((드래곤 피어 피해량\*0.3)+(자연의 공명 레벨\*2))\*스킬 피해량 스텟

&#x20;

4스킬 데들리 자밸린\
\- 전방으로 자밸린을 투척합니다\
\- 짧은 시간 동안 대상을 기절시킵니다\
\- 피해량 계수 : 피해량\*스킬 피해량 스텟\
\- 레벨당 피해량 : +20\
\- 레벨당 기절 지속시간: +0.1

&#x20;

5스킬 드래곤 피어(궁극기)\
&#x20; \- 해당 스킬의 지속시간동안 자연의 공명 스킬 계열의 콤보를 사용 불가능하게 변동합니다.\
&#x20; \- 플레이어의 활에 드래곤의 혼이 깃듭니다\
&#x20; \- 이후 전방으로 화염의 활을 발사하며 발사 된 이후 다시 드래곤 피어를 사용 할 경우 약화된 드래곤 피어를 5회 추가 사용 할 수 있습니다. (해당 효과는 속사 사용 시 즉시 종료 됩니다.)\
&#x20; \- 드래곤 피어 1타 피해량 계수 : (피해량\*0.8)\*스킬 피해량 스텟\
&#x20; \- 드래곤 피어 후속타 피해량 계수 : (피해량\*0.5)\*스킬 피해량 스텟\
&#x20; \- 레벨당 피해량 : +50
