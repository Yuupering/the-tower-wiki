# 개화



타락의 힘\
&#x20; -최대 체력 -1000 최대 마나+500\
&#x20; -팔라딘의 최대 체력에 비례해 모든 스킬의 최종 피해량이 증가됩니다\
&#x20; -최대 체력 100당 피해량이 10%씩 증가됩니다.\
&#x20; -디바인 스피릿과 기존 신성한 힘의 아군 회복이 삭제됩니다. \
&#x20; -디바인 스피릿과 디바인 월의 자신 회복이 삭제됩니다. \
&#x20; -신성한 힘의 회복이 기존 회복의 25%로 변경됩니다.\
&#x20; -홀리 스트라이크의 일반몬스터 최대 체력 비례 피해가 삭제됩니다.



종횡무진\
&#x20; -무기 공격력 7.5 물리 치명타 확률 +3% 물리 피해량 -5%\
&#x20; -개화 착용 시 전방 도약 스킬이 변화됩니다.\
&#x20; -전방 도약 스킬 사용 시 쉴드 효과를 얻습니다.\
&#x20; -지속 시간 : 전방 도약 레벨 × 0.05초\
&#x20; -쉴드 수치: 5+3((장착 중인 무기 공격력 + 전방 도약 레벨 x2)/10)\
&#x20; -도약 스킬 사용 이후 주변 5칸에 장착 중인 무기 공격력과 전방 도약 레벨에 비례해 피해를 입힙니다. \
&#x20; -신성한 힘 스킬의 저항 레벨이 감소됩니다.\
&#x20; -타락의 힘 착용 시 피해량 계산식이 변동됩니다.



#### 개화는 하늘섬 랜덤박스존의 Special Coin Shop 당나귀 npc에게서 구매할수 있습니다.
