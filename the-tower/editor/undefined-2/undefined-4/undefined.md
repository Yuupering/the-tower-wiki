# 개화



인파이트 거너\
&#x20; -최대 체력 +300 최대 마나 -200\
&#x20; -개화 스킬 장착시 일부 스킬이 근거리로 변경됩니다.\
&#x20; -피스톨 샷의 최대 사거리가 30에서 6로 감소됩니다.\
&#x20; -피스톨 샷의 피해량이 2.2배 증가됩니다.\
&#x20; -삿건 사격의 최대 사거리가 15에서 6로 감소됩니다.\
&#x20; -삿건 사격의 피해량이 1.5배 증가됩니다.\
&#x20; -데드아이 패시브의 모든 스킬 피해량 증가 스텟이 0.5 증가됩니다.\
&#x20; -백스텝 스킬이 전방으로 사용됩니다.



데스 리볼버\
&#x20; -스킬 치명타 추가 피해 +5% 최대체력 -300 최대 마나 +250 스킬 쿨타임 감소 -5%\
&#x20; -백업 스나이퍼 사용시 죽음의 탄환 4발을 장전합니다.\
&#x20; -데스 리볼버 - 스팀팩 (발동 조건: 죽음의 탄환 장전 이후 피스톨 샷)\
&#x20; -스팀팩 사용시 마나 대신 본인의 최대 체력 34%를 사용하며\
&#x20; -15초동안 스킬 피해량 13%, 스킬 쿨타임 감소 3.3% 버프를 얻습니다.\
&#x20; -데스 리볼버 - 죽음의 세레나데 (발동 조건: 죽음의 탄환 장전 이후 죽음의 난사)\
&#x20; -죽음의 세레나데 사용시 마나 대신 본인의 최대 체력 25%를 사용하며\
&#x20; -전방으로 도약 이후 죽음의 난사의 계수에 비례해 주변 10칸에 4번의 피해를 입힙니다.
