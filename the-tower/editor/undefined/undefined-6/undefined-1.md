# 양조 단계 레시피

공통 사항!

각 타입이 제작된 이후 하급 촉매제를 우클릭 하시면 타입 제조가 완성이 됩니다. 이후 각 타입에 맞는 용액을 우클릭하시면 제조가 완성 되며 완성품을 양조병으로 수거 할 수 있습니다.\
이후, 중급 촉매제, 상급 촉매제, 최상급 촉매제를 단계별로 클릭하시면 포션의 등급이 상승됩니다.



Type A 스킬 피해량 증가, 마나 회복의 포션, 빛의 포션을 제작하는데 사용됩니다.\
재료 : 밀 x 10, 사탕 수수 x 2, 레드스톤 블럭 x 3\
약불 : 10 \~ 20초 이후 하급 촉매제 우클릭



Type B 더블 점프, 치유의 노래, 광전사의 포션, 피해 감소의 포션\
재료 : 오징어 먹물 x 3, 썩은 살점 x 3, 독이 든 감자 x 7, 레드스톤 블럭 x 3\
중불 : 17 \~ 25초 이후 하급 촉매제 우클릭



Type C 스킬 쿨타임 감소, 번개의 포션, 마나 증폭의 포션\
재료 : 네더 사마귀 x 5, 꿀이 든 병 x 2, 레드스톤 블럭 x 3\
강불 : 17 \~ 25초 이후 하급 촉매제 우클릭



Type D 체력 회복의 포션, 신속의 포션, 체력 증폭의 포션\
재료 : 당근 x 5, 익히지 않은 소고기 x 5, 익히지 않은 닭고기 x 2, 레드스톤 블럭 x 3\
약불 : 7 \~ 15초 중불 : 7 \~ 15초 강불 : 7 \~ 15초 이후 하급 촉매제 우클릭



현자의 돌 레시피 (현자의 돌 전용 연금술에서만 시도 가능합니다.)\
재료 : 제련된 문스톤 x 5, 제련된 블러디스톤 x 5, 제련된 오리하르콘 x 5, 네더의 별 x 5\
약불 : 28초 \~ 33초 중불 : 28초 \~ 33초 강불 : 28 \~ 33초\
확률 : 10%의 확률로 성공. \
**실패 할 경우 재료가 소멸됩니다.**\
재료 넣기 -> 가열 -> 불끄기(풀무 쉬프트+우클릭) -> 최상급 촉매제 우클릭 -> 양조병으로 수거
