# 농사

<figure><img src="../../../.gitbook/assets/image (41).png" alt=""><figcaption><p>&#x3C;추천설치 예시></p></figcaption></figure>

해당 구조처럼 건설하시면 까마귀가 작물을 훔쳐가지 않습니다. (허수아비가 까마귀에 의한 작물 손상을 방지합니다.)

뼛가루는 사용이 금지되며 우클릭시 수확이 가능합니다.\
\
확률에 의해 수확시 등급이 결정됩니다.\
일반 등급의 작물은 '[<mark style="color:yellow;">**요리**</mark>](undefined-5/)'로도 사용이 가능하며 금등급 작물은 '[<mark style="color:yellow;">**도감**</mark>](../undefined-3/)' 에 등록이 가능합니다.\
작물마다 자라는 단계, 수확시 드롭 갯수, 판매 가격이 다릅니다.

또한, 커스텀 작물 수확 시, 2% 확률로 대지의 기운이 담긴 열매를 추가 획득하실 수 있습니다.\
농부 특성을 보유한 경우 10%로 증가합니다.\
(해당 아이템은 은빛주괴 교환 및 강렬한 야생의 기운 조합에 활용됩니다.)\
\
작물들은 다음과 같은 확률로 등급이 결정됩니다.\
일반 : 89.9%, 은 : 10%, 금 : 0.1%

토마토 : 1\~8개 수확 가능, 0.1% 확률로 거대화 작물까지 성장합니다.\
포도 : 1\~5개 수확 가능, 수확 후 나무가 남아 씨앗을 다시 심을 필요가 없습니다.\
홉 : 1\~7개 수확 가능, 수확 후 나무가 남아 씨앗을 다시 심을 필요가 없습니다.\
용과 : 1\~6개 수확 가능, 수확 후 나무가 남아 씨앗을 다시 심을 필요가 없습니다.\
마늘 : 2\~3개 수확 가능\
가지 : 1\~4개 수확 가능\
고추 : 1\~4개 수확 가능, 수확 후 나무가 남아 씨앗을 다시 심을 필요가 없습니다.\
청경채 : 1\~2개 수확 가능, 땅이 말라있는경우 성장하지 않습니다.\
양배추 : 1\~2개 수확 가능, 0.1% 확률로 거대화 작물까지 성장합니다.\
옥수수 : 1\~3개 수확 가능\
파인애플 : 1\~5개 수확 가능, 0.1% 확률로 거대화 작물까지 성장합니다.\
\
특수 농사 땅은 최대 갯수 사용 제한이 있습니다.\
7x7 2개까지 사용가능하며, 상한선은 농부업적으로만 올릴 수 있습니다.\
농부 업적 전체 클리어 시, 해당 징표를 장식하여 7x7의 농사 땅을 1개 더 증가 시킬 수 있습니다.\
(농부 특성일때만 도전과제 도전이 가능하며, 징표는 농부 특성이 아니어도 사용가능 합니다.)\
농사 땅은 농사 업적 클리어 보상 외에는 더 이상 늘릴수 없습니다.\
타 인원의 농사땅을 대신 캘수 없습니다.
