# 광산

<figure><img src="../../../.gitbook/assets/2024-10-30_20.59.50.png" alt=""><figcaption><p>&#x3C;The Tower의 광산입니다.></p></figcaption></figure>

3분에 한번씩 광물이 확률에 의해서 재 생성됩니다.\
언제 스폰되는지 공지가 되지 않습니다.\
광물은 따로 생성 되었다고 표기가 되지 않습니다.\
광물이 없는 위치에만 광물이 확률적으로 생성됩니다.

\
확률표 :\
레드스톤 4%\
석영 24%\
금 14%\
철 15%\
구리 21%\
석탄 4%\
청금석 13%\
다이아몬드 2%\
에메랄드 2%\
고대 잔해 1%
