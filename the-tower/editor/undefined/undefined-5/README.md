등급에 맞는 이용권을 구매하여 사용하시면 10분동안 이용하실 수 있습니다.\
(높은 등급을 사용하시면 하위등급 요리까지 전부 이용가능합니다)

부싯돌과 부시를 우클릭하여 요리 솥을 활성화 할 수 있습니다.

모든 요리는 <mark style="color:yellow;">**"요리사의 열정"**</mark> 아이템을 소비하여 제작이 가능합니다.

<mark style="color:yellow;">**"요리사의 열정"**</mark> 아이템은 야생의 특수 몬스터를 잡으면 확률로 습득 할 수 있습니다.

각 등급의 맞는 요리사의 열정과 재료 3개를 요리 솥을 우 클릭을 이용하여 넣은 이후\
부싯돌과 부시를 우클릭 하면 요리 제작을 시도합니다.

요리 레시피가 틀릴경우 음식물 쓰레기가 나옵니다.

모든 요리는 개인이 섭취하거나 <mark style="color:yellow;">**"도감"**</mark> 에 등록할 수 있습니다.

요리는 일반, 희귀, 전설, 신화 네가지 등급이 있습니다.
