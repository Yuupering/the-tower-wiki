# 일반 등급 요리 레시피

토마토 주스 = \[일반] 요리사의 열정 + 토마토 + 설탕 + 물 양동이(요리아이템)

오렌지 주스 = \[일반] 요리사의 열정 + 오렌지 + 설탕 + 물 양동이(요리아이템)

포도 주스 = \[일반] 요리사의 열정 + 포도 + 설탕 + 물 양동이(요리아이템)

체리 주스 = \[일반] 요리사의 열정 + 체리 + 설탕 + 물 양동이(요리아이템)

바나나 우유 = \[일반] 요리사의 열정 + 바나나 + 꿀 + 우유

딸기 우유 = \[일반] 요리사의 열정 + 딸기 + 꿀 + 우유

초콜릿 우유 = \[일반] 요리사의 열정 + 코코아 + 설탕 + 우유

애플 캔디 = \[일반] 요리사의 열정 + 사과 + 막대기 + 꿀

맥주 = \[일반] 요리사의 열정 + 홉 + 알코올 + 탄산수

팝콘 = \[일반] 요리사의 열정 + 옥수수 + 옥수수 + 버터

와인 = \[일반] 요리사의 열정 + 포도 + 설탕 + 알코올

미역국 = \[일반] 요리사의 열정 + 말린켈프 + 소금 + 물 양동이(요리아이템)

계란 후라이 = \[일반] 요리사의 열정 + 달걀 + 소금 + 기름

초콜릿 바 = \[일반] 요리사의 열정 + 코코아 + 설탕 + 빵

도미 스시 = \[일반] 요리사의 열정 + 도미 + 쌀 + 소금

연어 스시 = \[일반] 요리사의 열정 + 연어 + 쌀 + 소금

롤리팝 사탕 = \[일반] 요리사의 열정 + 딸기 + 설탕 + 막대기

솜사탕 = \[일반] 요리사의 열정 + 설탕 + 설탕 + 막대기

아이스 커피 = \[일반] 요리사의 열정 + 커피 콩 + 커피 콩 + 물 양동이(요리아이템)

샌드위치 = \[일반] 요리사의 열정 + 빵 + 양배추 + 토마토

도넛 = \[일반] 요리사의 열정 + 빵 + 기름 + 설탕

토마토 파스타 = \[일반] 요리사의 열정 + 토마토 + 밀 + 돼지고기

로제 파스타 = \[일반] 요리사의 열정 + 토마토 + 밀 + 크림

크림 파스타 = \[일반] 요리사의 열정 + 마늘 + 밀 + 크림

팬케이크 = \[일반] 요리사의 열정 + 버터 + 빵 + 꿀
