# 거주지

거주지로 이동하면, 1, 2구역의 거주구역으로 이동하실 수 있는 표지판이 있습니다.

±2000, ±2000에 각 4개, ±6000, ±6000에 각 4개의 [달빛 제단](../../editor/undefined/undefined-4.md)이 존재합니다.

네더 포탈을 생성하여 네더월드로 가실 수 있으며, 네더 월드의 0, 0 좌표에는 [네더 제단](../../editor/undefined/undefined-4.md)이 존재합니다.



많은 유저들의 집들과 건축물이 자유롭게 설치된 구역입니다.

또한, 여러 특수 던전 역시 존재하여, 서버 측 에서 건축한 던전 및 구조물은 지형 보호가 되어 있습니다.
